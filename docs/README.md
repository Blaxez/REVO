# Documentation

This directory contains images and documentation for the REVO extension.

## Preview Image

The [popup-preview.png](popup-preview.png) file should be a screenshot of the extension's popup interface.
This image is referenced in the main README.md file.

To create this image:

1. Install the extension in Chrome
2. Open the popup interface
3. Take a screenshot
4. Save it as `popup-preview.png` in this directory
